## 二、翻译原则

### 2.1 白话标准

以老舍、赵树理等现当代作家的叙事白话为基准，做到：

- **能读**：一句读完不卡壳，不需要反复看
- **能懂**：不夹带文言词汇和句式
- **能讲**：可以像讲故事一样顺畅地念出来

### 2.2 具体方法

#### ① 句式转换

| 文言 | → | 白话 |
|------|---|------|
| ○甲戌/青城伯沈德符卒。德符字得之... | → | 青城伯沈德符去世了。沈德符，字得之…… |
| ○上闻之曰："此必闵无咎嗾宪司为之也。" | → | 太宗听说后说："这肯定是闵无咎唆使宪司干的。" |
| ○月犯毕星。 | → | 晚上，月亮运行到了毕宿附近。 |

#### ② 消除"翻译腔"

不要出现以下表达：

- ❌ "××日/太宗前往太上殿朝见，又至上王殿行贺礼"
- ✅ "大年初一，太宗先去太上殿给太上王朝贺，又去上王殿行礼"

核心原则：**先按现代汉语想一遍这句话该怎么说，再落笔**

#### ③ 人名官名处理

- 关键人名（太宗、太上王、河崙、李茂等）直接使用
- 生僻官名在行文中自然融入，不单独加注
  - ❌ "任命检校参赞门下府事权钧为..."
  - ✅ "参赞门下府事权近上了一道长长的奏折"
- 过于冗长的古代官名可简化为"大臣""官员""监察官"等

#### ④ 天文灾异条目

原文中大量"太白昼见""雷""地震""雨雹"等简略记录，在白话文中扩写为自然叙述：

- "金星大白天就看得见"
- "下冰雹了，有弹丸那么大"
- "打雷了，还地震了"

#### ⑤ 长篇奏疏

大臣的长篇奏疏（尤其是权近、柳观等人的）在白话文中做概括处理：保留核心观点和论据，去除连篇累牍的引经据典。

## 三、文言⇄白话对比样本

### 样本1：日常记事

**原文：**
> 辛酉朔/上朝太上殿, 又至上王殿行贺礼。太上王适幸神岩寺, 故但进绮绢表里而还。

**旧译（翻译腔）：**
> 辛酉朔日/太宗前往太上殿朝见，又到上王殿行贺礼。太上王恰好去了神岩寺，所以只进献了绮绢表里后返回。

**新译（白话文）：**
> 大年初一，太宗先去太上殿给太上王朝贺，又去上王殿行礼。太上王正好去了神岩寺，所以只送了绸缎礼品就回来了。

### 样本2：人物传记

**原文：**
> 德符字得之, 青鳧县人, 典理正郞龙之子也。以门荫, 始受左右卫录事参军, 累迁少府尹。恭愍王十三年甲辰, 出守水原府, 廉使至府, 德符纳谒, 见使不裩便退, 使命吏让之, 对以服不如仪, 使谢不敏。

**旧译（逐字直译）：**
> 沈德符字得之，青凫县人，典理正郎沈龙之子。以门荫入仕，起初任左右卫录事参军，累迁至少府尹。恭愍王十三年甲辰，出任水原府守，廉使到府，德符谒见，见廉使不穿裤子就退了出来，廉使命官吏责备他，他回答说是服仪不合规定，廉使道歉说自己不敏锐。

**新译（白话文）：**
> 沈德符字得之，青凫县人，他父亲沈龙当过典理正郎。他靠父辈的关系进了官场，一开始做左右卫录事参军，慢慢升到了少府尹。高丽恭愍王十三年，他被派到水原府当地方官。有一次上级监察官来视察，沈德符去拜见，发现那位大人连裤子都没穿，转身就退了出来。监察官派人责备他，他说自己的衣服不合礼仪，监察官反倒道歉说自己考虑不周。

### 样本3：天象记录

| 原文 | 旧译 | 新译 |
|------|------|------|
| 月犯毕星 | 月亮侵犯毕星 | 晚上，月亮运行到了毕宿附近 |
| 太白昼见 | 金星白天出现 | 金星大白天就看得见 |
| 木稼 | 树上结冰如禾稼 | 树上结了厚厚的冰凌 |
| 日珥 | 出现日珥 | 太阳旁边出现了日珥 |
| 雨雹大如弹丸 | 下冰雹大如弹丸 | 下冰雹了，有弹丸那么大 |

### 样本4：短条目统一处理

| 文言 | 白话译文 |
|------|---------|
| 雷 | 打雷了 |
| 地震 | 地震了 |
| 陨霜 | 降霜了 |
| 雨 | 下雨了 |
| 雾 | 起雾了（大雾弥漫） |
| 昏雾四塞 | 大雾弥漫，对面不见人 |
| 星入太微 | 有流星划过天空 |

## 四、翻译节奏

- **短条目**（天象、任命、进贡等）：1~2句，干净利落
- **中等条目**（政令、人事纠纷、小型典礼）：3~5句，把事情说清楚
- **长叙事**（廷争、使臣往来、重要事件、人物传记）：完整展开
- **长篇奏疏**：概括核心论点，保留关键论据

## 五、质量检查清单

每条译文完成后自查：

- [ ] 能用"__然后__""__于是__""__原来__"等口语连接词自然地串起来？
- [ ] 去掉"○干支"格式后读起来像现代文章？
- [ ] 一个句子超过30个字就考虑拆分？
- [ ] "之乎者也"全去掉了？
- [ ] 有没有留下"XX日/太宗XX"这种翻译腔句式？
- [ ] 人物对话用了引号，读起来像在说人话？
